API TESTING CON RESTASSURED

* PROYECTO ELEGIDO: OPCION#3

-   El proyecto fue diseñado, desarrollado y ejecutado utilizando Maven, Java, RestAssured, TestNG y Allure.
    Se utilizó el patrón Data Factory Builder.

-   Para ejecutar la Test Suite, desde IntelliJ, en la solapa Maven - Test podemos ejecutar todos los tests.
    Automáticamente se generará la carpeta "Allure Results", que contiene todos los datos resultantes de nuestras
    pruebas.
    Desde la consola, ubicados en el directorio del proyecto, con el commando "Allure serve", seremos redirigidos a
    una página Html con el dashboard de Allure, el cual nos muestra de forma gráfica y clara los resultados
    de las pruebas ejecutadas.
